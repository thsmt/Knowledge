Knowledge

CloudFormation、AWS CLI、Terraform、AWX / Ansible など、クラウドエンジニアの実務で使う手順を横断的に整理するためのドキュメントサイトです。

各GitHubリポジトリには構成図、CloudFormationテンプレート、パラメータ例を置き、このサイトにはリポジトリをまたいで使う運用手順や確認観点をまとめます。

📍 運用方針

リポジトリは実装本体、Knowledge は横断手順集として分けます。変更作業、障害時の確認、ツール操作の手順を中心に整理します。

手順一覧

CloudFormation 運用手順集

デプロイ方法、変更セット、ロールバック、ドリフト検出、削除前確認などを整理します。

AWS CLI 特殊対応

CLI/APIでしか変更・切り戻し・復旧できない運用手順をまとめます。

AWS CLI コマンドリファレンス

認証確認、Security Group、Prefix List、SNSなど、日常運用で使う基本コマンドを整理します。

障害切り分けチェックリスト

Linux障害発生時の初動記録、ログ、リソース、ネットワーク、サービス状態の確認観点を整理します。

Amazon WorkSpaces

WorkSpacesのディレクトリ登録、Web Access、カスタムイメージ、Cost Optimizer、トラブル対応を整理します。

Amazon SES

カスタムMAIL FROM、Route 53レコード、PostfixからSES SMTPへのリレー、送信テストを整理します。

Terraform

init、plan、apply、destroy、tfstate、backend、variables などを実務目線で整理します。

[構築] Red Hat Enterprise Linux 9

基本パッケージ導入、logrotate設定、ミドルウェア別logrotate設定の注意点を整理します。

[構築] Amazon Linux 2023

Amazon Linux 2023の初期設定、追加パッケージ、rsyslog、cron、リリース固定を整理します。

[構築] Windows Server 2022

EC2起動、RDP接続、ローカルユーザー作成、OS初期設定、Windows Updateの確認観点を整理します。

Zabbix 6.0 運用手順

Zabbix Agent、ホスト登録、監視項目、メンテナンス設定、作業前後エビデンス取得の観点を整理します。

AWX / Ansible 手順

Inventory、Credential、Project、Job Template、Playbook実行ログの確認観点を整理します。

EOL / ライフサイクル一覧表

OS、AWSサービス、ミドルウェア、エージェントのサポート期限と確認リンクを整理します。
